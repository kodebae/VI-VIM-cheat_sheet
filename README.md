# VI-VIM-cheat_sheet
VI / VIM Cheat Sheet

## Intro
Vim works in modes, in normal modes keys have normal functionalities.

- insert mode
```
i
```
- exit insert mode
```
<esc>
```
-How to get out of vim
```
:q + <Enter>
```
- quit and dismiss all changes
```
:q! + <Enter>
```
-write changes
```
:w
```
